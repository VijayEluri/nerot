Nerot

Scheduled Caching Made Simple

Add this dependency to your pom.xml: 
<dependency>
    <groupId>nerot</groupId>
    <artifactId>nerot</artifactId>
    <version>3.0</version>
</dependency>
Add this repository to your pom.xml: 
<repository>
    <id>nerot-repository</id>
    <name>Nerot Repository</name>
    <url>http://garysweaver.github.com/nerot/m2/releases</url>
</repository>
Include nerot.xml in your web.xml (or add its contents into your spring config): 
<web-app>
    ...
    <context-param>
        <param-name>contextConfigLocation</param-name>
        <param-value>classpath:nerot.xml
                     ...
        </param-value>
    </context-param>
    ...
</web-app>
Add Nerot to your class:

import nerot.Nerot;

public class SomeClass {
  
  private Nerot nerot;
                
  public void setNerot(Nerot nerot) {
    this.nerot = nerot;
  }
}
Set Nerot on your bean: 
<bean name="yourController" class="com.acme.YourController">
    <property name="nerot" ref="nerot"/>
    ...
</bean>

Familiar Scheduling Syntax

"0/5 * * * * ?"

Nerot uses Quartz CronTrigger Syntax, which is similar to the familiar cron syntax.

RSS feed retrieval

// Schedule on initialization
nerot.scheduleRss(url, "0 * * * * ?");

// Retrieve multiple times without delay
nerot.getRssFromStore(url);

HTTP GET

// Schedule on initialization
nerot.scheduleHttpGet(url, "0 * * * * ?");

// Retrieve multiple times without delay
nerot.getHttpResponseBodyFromStore(url);

Static Methods

// Schedule on initialization
GenericTask task = new GenericTask();
task.setStoreKey(key);
task.setActor(Math.class);
task.setMethod("random");
nerot.schedule("myJob", task, "0 * * * * ?");

// Retrieve multiple times without delay
Double result = (Double)nerot.getResultFromStore(key);

Instance Methods

// Schedule on initialization
GenericTask task = new GenericTask();
task.setStoreKey(key);
task.setActor(new Foobar());
task.setMethod("someFantasticMethod");
task.setArgs(new Object[]{"s", y, true, 'c', 1.0D});
nerot.schedule("myJob", task, "0/1 * * * * ?");

// Retrieve multiple times without delay
Foobar result = (Foobar)nerot.getResultFromStore(key);
